# Python Full Stack Shopping Application

## Project Overview

This project is a full-stack e-commerce web application developed using React, FastAPI, Python, and MySQL. The application allows users to register, log in, browse products, and place orders through a user-friendly interface. The backend provides RESTful APIs for handling business logic and database operations, while MySQL stores user, product, and order information securely.

## Features

* User Registration and Login
* Product Management
* Order Management
* REST API Integration
* MySQL Database Connectivity
* Responsive User Interface
* Frontend and Backend Integration
* CRUD Operations

## Installation

### Clone Repository

git clone <repository-url>

### Backend Setup

cd backend

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

uvicorn app.main:app --reload

### Frontend Setup

cd frontend

npm install

npm start
